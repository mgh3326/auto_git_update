# github master 자동으로 올려주기
> github master 자동으로 올려주기

<hr/>

my.sh 파일을 remote 저장소가 지정된 git 폴더로 이동 시킨후에 git bash 에서 ./my.sh (커밋 내용) 을 실행할 경우
git pull origin master
git add *
git commit (커밋 내용)
git push origin master 를 진행하게 됩니다.

커밋 내용에 ""은 필요 없으며, 공백이 들어가도 무방합니다.

감사합니다.
